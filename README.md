## Deskripsi Program
Program ini merupakan tugas Ujian Akhir Semester mata kuliah Komputer Grafika
yang menampilkan objek Kubus 3D dan Persegi 2D dalam satu window menggunakan
Pygame dan PyOpenGL.

## Fitur Program
- Kubus 3D dengan transformasi translasi, rotasi, dan skala
- Persegi 2D dengan transformasi translasi, rotasi, skala, shearing, dan refleksi
- Tampilan dua objek dalam satu window menggunakan viewport

## Cara Menjalankan Program
1. Install library yang dibutuhkan:
   pip install pygame PyOpenGL PyOpenGL_accelerate
2. Jalankan program dengan perintah:
   python main.py
